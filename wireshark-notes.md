# Wireshark Basics 🧢

## What is Wireshark?
Wireshark is a tool used to capture and analyze network traffic in real time.

It helps you see what is happening inside a network.

## What can it do?
- Capture live network packets
- Analyze internet traffic
- See communication between devices
- Understand protocols (HTTP, DNS, TCP)

## How it works:
Wireshark shows data in "packets" which are small pieces of information sent over a network.

## Basic Use:
1. Open Wireshark
2. Select a network interface (Wi-Fi or Ethernet)
3. Start capture
4. Watch packets appear in real time

## What to look at:
- Source IP (who is sending)
- Destination IP (who is receiving)
- Protocol (how data is sent)
- Info (what kind of data)

## Important Note:
Only capture traffic on your own network or with permission.
