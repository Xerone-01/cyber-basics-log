# Attack Surface Basics 🧢

## What is Attack Surface?
Attack surface is all the possible points where a system can be accessed or attacked.

## Examples:
- Open ports (like 22, 80, 443)
- Web applications
- APIs
- Network services
- User inputs

## Why it matters:
The more exposed services a system has, the larger its attack surface.

## How tools help:
- Nmap helps discover open ports
- Wireshark helps analyze network traffic

---

## Goal:
Understand what parts of a system are visible and potentially exposed.
